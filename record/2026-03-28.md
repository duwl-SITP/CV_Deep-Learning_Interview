## 1. 介绍Transformer中使用的激活函数

在 Transformer 架构中，非线性激活函数主要应用于前馈神经网络（Feed-Forward Network, FFN）模块。随着大规模语言模型架构的演进，激活函数的选择呈现出从简单分段线性函数向平滑、非单调函数以及门控机制发展的趋势。

### 1. ReLU (Rectified Linear Unit)

ReLU 是 2017 年初代 Transformer 模型采用的基准激活函数。

- **数学定义**：
    
    $$\text{ReLU}(x) = \max(0, x)$$
    
- **数学特性**：对于输入 $x > 0$ 的区间，其导数恒为 1，有效缓解了深层网络反向传播中的梯度消失问题；且由于无需计算指数运算，硬件执行效率极高。
    
- **局限性**：当输入 $x \le 0$ 时，梯度严格为 0。若前向计算结果长期处于负区间，对应神经元的权重将无法获得梯度更新。此外，该函数在 $x = 0$ 处不可导。
    

### 2. GELU (Gaussian Error Linear Unit)

GELU 广泛应用于 BERT、GPT-2 及 Vision Transformer (ViT) 等架构中，是该演进阶段的标准配置。

- **数学定义**：
    
    $$\text{GELU}(x) = x \Phi(x)$$
    
    其中，$\Phi(x)$ 为标准正态分布的累积分布函数（CDF）。
    
- **数学特性**：GELU 将非线性映射与随机正则化特性相结合。其函数图像处处连续且无限次可导。在 $x$ 略小于 0 的区间内，GELU 允许输出微小的负值，这种非单调性使得模型能够保留部分有界的负向响应，扩展了特征表达的维度。
- **缺点**：**计算开销较高**。其严格数学定义依赖于计算复杂的误差函数（Error Function, $\text{erf}$）。即便采用三次多项式或双曲正切函数（Tanh）进行数值逼近，其浮点运算次数仍显著高于基础分段线性函数。

### 3. SiLU (Sigmoid Linear Unit) / Swish

SiLU 在数学形式上等价于参数 $\beta=1$ 时的 Swish 函数，目前被 LLaMA、Qwen 等多数开源大规模语言模型采用。

- **数学定义**：
    
    $$\text{SiLU}(x) = x \sigma(x) = \frac{x}{1 + e^{-x}}$$
    
- **数学特性**：SiLU 的函数曲线、平滑度及非单调下界特性与 GELU 高度近似，在模型收敛性及最终精度指标上表现出极高的等效性。
    
- **工程优势**：由于 SiLU 的解析式直接依赖于标准的 Logistic Sigmoid 函数，在现代 GPU 底层算子优化中，其执行效率通常优于需要计算误差函数近似值的 GELU。
    

### 4. SwiGLU (Swish-Gated Linear Unit)

SwiGLU 本质上是对传统 FFN 模块的结构性重构，引入了门控机制，被 LLaMA 和 PaLM 等模型作为核心组件。

- **数学定义**：
    
    $$\text{SwiGLU}(x, W, V) = \text{SiLU}(xW) \otimes (xV)$$
    
- **结构解析**：输入张量 $x$ 分别经历两次独立的线性映射（分别乘以权重矩阵 $W$ 和 $V$）。其中一个分支经过 SiLU 函数进行非线性激活，产生门控信号；随后与另一条线性分支进行逐元素乘法（Hadamard Product, $\otimes$）。
    
- **理论优势**：实证研究表明，在严格对齐参数总量与浮点运算次数（FLOPs）的条件下，采用 SwiGLU 结构的门控前馈网络能够在语言建模任务中实现更低的交叉熵损失与更优的泛化性能。

## 2. 手撕LayerNorm和BatchNorm

### A. 批量归一化 (Batch Normalization)

**步骤一：计算特征维度的经验均值（Empirical Mean）**

对于第 $j$ 个特征，计算当前批量内所有样本的均值：

$$\mu_j = \frac{1}{N} \sum_{i=1}^N x_{i,j}$$

**步骤二：计算特征维度的经验方差（Empirical Variance）**

对于第 $j$ 个特征，计算当前批量内所有样本的方差：

$$\sigma_j^2 = \frac{1}{N} \sum_{i=1}^N (x_{i,j} - \mu_j)^2$$

**步骤三：标准化（Standardization）**

利用均值和方差对输入值进行零均值、单位方差的标准化处理。其中 $\epsilon$ 为防止分母为零的极小正常数（通常取 $10^{-5}$）：

$$\hat{x}_{i,j} = \frac{x_{i,j} - \mu_j}{\sqrt{\sigma_j^2 + \epsilon}}$$

**步骤四：仿射变换（Affine Transformation）**

引入可学习的缩放参数 $\gamma \in \mathbb{R}^D$ 与平移参数 $\beta \in \mathbb{R}^D$，以恢复网络对特征分布的非线性表达能力：

$$y_{i,j} = \gamma_j \hat{x}_{i,j} + \beta_j$$

### B. 层归一化 (Layer Normalization)

层归一化的核心运算逻辑是在特征维度（Feature Dimension, $D$）上对每一个独立样本计算统计量。

**步骤一：计算单一样本的均值**

对于第 $i$ 个样本，计算其所有特征值的均值：

$$\mu_i = \frac{1}{D} \sum_{j=1}^D x_{i,j}$$

**步骤二：计算单一样本的方差**

对于第 $i$ 个样本，计算其所有特征值的方差：

$$\sigma_i^2 = \frac{1}{D} \sum_{j=1}^D (x_{i,j} - \mu_i)^2$$

**步骤三：标准化**

对当前样本的特征进行标准化处理：

$$\hat{x}_{i,j} = \frac{x_{i,j} - \mu_i}{\sqrt{\sigma_i^2 + \epsilon}}$$

**步骤四：仿射变换**

引入可学习的缩放参数 $\gamma \in \mathbb{R}^D$ 与平移参数 $\beta \in \mathbb{R}^D$（参数在所有样本间共享）：

	$$y_{i,j} = \gamma_j \hat{x}_{i,j} + \beta_j$$

## 3. 介绍Safe Softmax

### A. 标准 Softmax 的数值不稳定性根源

给定输入向量 $x \in \mathbb{R}^N$，标准 Softmax 函数的第 $i$ 个分量的计算公式为：

$$S(x_i) = \frac{e^{x_i}}{\sum_{j=1}^N e^{x_j}}$$

该公式在理论数学域中完全成立，但在计算机浮点数表示系（如 IEEE 754 标准的 FP32 或 FP16）中存在严重的数值缺陷：

- **上溢（Overflow）**：指数函数 $e^{x_i}$ 的增长速率极快。在单精度浮点数（FP32）下，当 $x_i \approx 88.72$ 时，$e^{x_i}$ 的值将超出 FP32 的最大表示范围（约 $3.4 \times 10^{38}$），系统将其计算为正无穷大（`inf`）。这会导致后续的分母求和以及除法运算均输出非数（`NaN`, Not a Number）。
    
- **下溢（Underflow）**：当输入向量的所有元素均为绝对值极大的负数时，所有的 $e^{x_i}$ 均会因超出浮点数的最小精度限制而被截断为 $0$。此时分母 $\sum_{j=1}^N e^{x_j} = 0$，触发除零错误，同样输出 `NaN`。

### B. Safe Softmax 的代数转换与等价性证明

Safe Softmax 通过平移输入向量的分布中心来消除指数爆炸的可能性。其核心操作是在计算指数之前，从输入向量的所有元素中减去该向量的最大值。

**算法步骤**：

1. 计算输入向量的全局最大值：$m = \max(x)$
    
2. 对所有元素执行常数减法：$x_i' = x_i - m$
    
3. 计算转换后的 Softmax：
$$S_{safe}(x_i) = \frac{e^{x_i - m}}{\sum_{j=1}^N e^{x_j - m}}$$

**严格代数等价性证明**：

根据指数函数的代数性质 $e^{A-B} = e^A \cdot e^{-B}$，对 Safe Softmax 的表达式进行展开：

$$S_{safe}(x_i) = \frac{e^{x_i} \cdot e^{-m}}{\sum_{j=1}^N (e^{x_j} \cdot e^{-m})}$$

提取分母中的公因式 $e^{-m}$：

$$S_{safe}(x_i) = \frac{e^{-m} \cdot e^{x_i}}{e^{-m} \cdot \sum_{j=1}^N e^{x_j}}$$

由于 $e^{-m} \neq 0$，分子与分母中的 $e^{-m}$ 严格抵消：

$$S_{safe}(x_i) = \frac{e^{x_i}}{\sum_{j=1}^N e^{x_j}} = S(x_i)$$

证明完毕。该转换在实数域内具有绝对等价性。

## 4. 梯度消失与梯度爆炸的解决方案

### 梯度消失的解决方案

- **残差连接 (Residual Connections)**：通过引入跨层恒等映射 $x_{l+1} = x_l + \mathcal{F}(x_l, W_l)$，在计算反向导数 $\frac{\partial x_{l+1}}{\partial x_l}$ 时强制引入常数单位矩阵 $I$。此结构确保了高层梯度向量能够以加性叠加的方式无损反向传递至底层网络。
    
- **非饱和激活函数 (Non-saturating Activation Functions)**：采用 ReLU、GELU 或 SiLU 等函数替代 Sigmoid 与 Tanh。此类激活函数在其正向定义域内具有恒定为 1 或接近于 1 的导数，从代数定义上排除了因激活函数导数处于 $(0, 0.25]$ 区间而引发的连乘衰减效应。
    
- **方差恒定初始化 (Variance-Preserving Initialization)**：应用 Xavier 或 He 初始化策略。通过约束初始权重矩阵的方差与网络输入及输出维度的反比关系（如 $\text{Var}(W) = \frac{2}{n_{in}}$），确保前向传播中特征的方差与反向传播中梯度的方差在各层之间保持数值稳定，避免初始梯度的数量级塌陷。
    
- **前置层归一化 (Pre-Layer Normalization)**：在 Transformer 等深层架构中，将归一化操作置于残差相加之前。该拓扑结构将主干计算图转化为纯粹的线性累加，彻底规避了归一化算子的雅可比矩阵对反向梯度流的缩放削弱。

### 梯度爆炸的解决方案

- **梯度范数裁剪 (Gradient Norm Clipping)**：在执行参数更新步骤之前，计算当前全网梯度向量的全局 L2 范数 $\|g\|$。若该范数超出预设的安全阈值 $c$，则强制对梯度向量应用标量缩放：$g \leftarrow g \cdot \frac{c}{\|g\|}$。此操作保持了目标函数的优化下降方向不变，但严格限制了参数更新在参数空间中的欧氏距离步长。
    
- **参数范数惩罚 (Weight Regularization/Decay)**：在目标函数中引入针对权重矩阵参数的 L1 或 L2 正则化惩罚项（如 $\frac{\lambda}{2} \|W\|_2^2$）。该代数约束在优化过程中持续迫使网络参数趋向于数值 0，从而间接限制了雅可比矩阵中各元素的绝对值大小，降低了连乘放大效应的概率。
    
- **批量/层归一化 (Batch/Layer Normalization)**：归一化算子不仅能缓解梯度消失，同样能抑制梯度爆炸。通过对中间层特征特征执行零均值与单位方差的强制约束，归一化层在代数上限制了隐藏状态矩阵 $H$ 的数值尺度，进而约束了依赖于 $H$ 进行计算的局部梯度矩阵的范数上限。
    
- **截断正态分布初始化 (Truncated Normal Initialization)**：在初始化权重参数时，如果采样的随机变量超出了设定的标准差范围（如 $[-2\sigma, 2\sigma]$），则强制重新采样。此机制从物理上排除了权重矩阵初始化时存在极端大数值元素的可能性，控制了初始迭代阶段的梯度量级。
